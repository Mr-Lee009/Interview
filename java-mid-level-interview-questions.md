# 10 Câu Hỏi Phỏng Vấn Java Mid-Level

1. Hãy giải thích sự khác nhau giữa `==` và `equals()` trong Java. Khi nào nên dùng từng cái?

2. `HashMap` hoạt động như thế nào? Điều gì xảy ra khi có collision?

3. Sự khác nhau giữa `ArrayList` và `LinkedList` là gì? Trong trường hợp nào nên chọn mỗi loại?

4. Hãy giải thích các khái niệm `encapsulation`, `inheritance`, `polymorphism`, và `abstraction` trong Java với ví dụ thực tế.

5. Sự khác nhau giữa `checked exception` và `unchecked exception` là gì? Khi nào nên tự tạo custom exception?

6. `String`, `StringBuilder`, và `StringBuffer` khác nhau như thế nào? Vì sao `String` là immutable?

7. Hãy mô tả vòng đời của một thread trong Java. `synchronized` dùng để làm gì?

8. `final`, `finally`, và `finalize` khác nhau như thế nào?

9. Java 8 mang lại những cải tiến nào nổi bật, đặc biệt là `Stream API` và `Lambda Expression`?

10. Trong một ứng dụng Java backend, bạn sẽ tối ưu hiệu năng như thế nào khi xử lý dữ liệu lớn hoặc truy cập database nhiều lần?
