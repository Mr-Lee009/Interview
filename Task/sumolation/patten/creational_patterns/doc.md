# Creational Patterns trong Java

Thư mục này đã được tách thành 5 file riêng, mỗi file gồm:

- Định nghĩa pattern
- Cách dùng
- Khi nào nên dùng
- Code Java mẫu
- Comment chi tiết ngay trong code

## Danh sách file

1. `singleton.md`
2. `factory_method.md`
3. `abstract_factory.md`
4. `builder.md`
5. `prototype.md`

## Tóm tắt nhanh

1. `Singleton`: cần duy nhất một instance dùng chung.
2. `Factory Method`: chọn class cụ thể để tạo object thông qua factory.
3. `Abstract Factory`: tạo cả một họ object liên quan với nhau.
4. `Builder`: tạo object phức tạp có nhiều tham số.
5. `Prototype`: tạo object mới bằng cách clone từ object mẫu.
