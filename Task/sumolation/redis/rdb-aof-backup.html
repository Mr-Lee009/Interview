Redis Persistence: RDB và AOF

Mô phỏng quá trình Redis lưu dữ liệu xuống disk: RDB tạo snapshot theo thời điểm, AOF ghi lại command để replay khi khôi phục.

Các bước đang mô phỏng

So sánh nhanh

RDB

  • Lưu snapshot tại một thời điểm.
  • File nhỏ, restore nhanh.
  • Có thể mất dữ liệu sau snapshot cuối.

AOF

  • Ghi lại command thay đổi dữ liệu.
  • Mất ít dữ liệu hơn tùy fsync.
  • File lớn hơn, cần rewrite định kỳ.
